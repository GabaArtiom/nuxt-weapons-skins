📋 Быстрый старт (3 шага):

1️⃣  Настрой .env файл:
   nano .env
   
   Укажи данные MySQL из конфига плагина WeaponPaints:
   - DB_HOST, DB_USER, DB_PASSWORD, DB_NAME
   
   Получи Steam API Key (опционально):
   - https://steamcommunity.com/dev/apikey
   - Добавь в STEAM_API_KEY=

2️⃣  Запусти dev сервер:
   npm run dev

3️⃣  Открой браузер:
   http://localhost:3000
