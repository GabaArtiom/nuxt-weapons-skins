# ✅ Чеклист реализованного функционала

## 🎯 Основной функционал (MVP)

- [x] **Nuxt 4 проект** — инициализирован с правильной структурой `app/`
- [x] **Steam авторизация** — OpenID 2.0 логин/логаут
- [x] **Steam Web API** — получение профиля (никнейм, аватар)
- [x] **MySQL подключение** — connection pool через mysql2
- [x] **Список оружий** — главная страница с категориями
- [x] **Выбор скинов** — страница выбора скина для каждого оружия
- [x] **Сохранение в БД** — INSERT/UPDATE в таблицу `wp_player_skins`
- [x] **Превью изображений** — загрузка из CSGO-API
