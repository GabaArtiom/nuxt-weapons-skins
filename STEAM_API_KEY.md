# Получение Steam Web API Key

Steam Web API Key нужен для получения информации о профиле пользователя (никнейм, аватар).

## Как получить ключ

1. **Войди в Steam аккаунт**
   - Открой https://steamcommunity.com
   - Войди в свой аккаунт

2. **Перейди на страницу регистрации ключа**
   - Открой https://steamcommunity.com/dev/apikey
   - Если не авторизован — войди

3. **Заполни форму**
   - **Domain Name:** укажи домен твоего сайта
     - Для локальной разработки: `localhost`
     - Для production: `yourdomain.com` (без http/https)
   - Согласись с условиями
   - Нажми "Register"

4. **Скопируй ключ**
   - После регистрации увидишь свой API Key
   - Это строка вида: `XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`
   - Скопируй её

5. **Добавь в .env файл**
   ```env
   STEAM_API_KEY=твой_ключ_здесь
   ```

## Важно

- ⚠️ **Не коммить ключ в git!** — файл `.env` уже в `.gitignore`
- 🔒 **Храни ключ в секрете** — не публикуй его нигде
- 🌐 **Один ключ на домен** — для каждого домена нужен свой ключ
- 🔄 **Можно перегенерировать** — если ключ утёк, создай новый на той же странице

## Что даёт API Key

С ключом сайт может:
- ✅ Получить никнейм пользователя
- ✅ Получить аватар
- ✅ Получить ссылку на профиль Steam
- ✅ Показать красивую карточку профиля вместо просто SteamID

Без ключа:
- ❌ Будет показываться только SteamID64
- ❌ Не будет аватара и никнейма
- ⚠️ Но авторизация и сохранение скинов **будут работать**!

## Лимиты

Steam Web API имеет лимиты:
- **100,000 запросов в день** — более чем достаточно для небольшого сайта
- Если превысишь — API вернёт ошибку 429 (Too Many Requests)

## Troubleshooting

### Ошибка "Invalid API Key"
- Проверь что ключ правильно скопирован в `.env`
- Убедись что нет лишних пробелов
- Перезапусти dev сервер после изменения `.env`

### Ошибка "Access Denied"
- Проверь что домен в настройках ключа совпадает с SITE_URL
- Для localhost используй именно `localhost`, а не `127.0.0.1`

### Профиль не загружается
- Проверь интернет соединение
- Убедись что Steam API доступен (иногда бывают сбои)
- Проверь консоль браузера на ошибки

## Альтернатива без API Key

Если не хочешь получать API ключ, сайт будет работать, но:
- Вместо никнейма будет показываться SteamID64
- Не будет аватара
- Всё остальное (авторизация, выбор скинов, сохранение) работает нормально

Просто оставь `STEAM_API_KEY` пустым в `.env`:
```env
STEAM_API_KEY=
```
